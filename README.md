# 🎉 Invito - Full Stack Event Management Website

**Invito** is a full-stack event management platform designed to simplify the process of hosting and managing events online. From personal parties to corporate meetups, Invito handles it all — with an intuitive UI and powerful backend integration.

## 🧰 Tech Stack

### Frontend:
- React JS
- Tailwind CSS
- React Router
- Framer Motion (for animations)
- Axios (API communication)

### Backend:
- Node.js
- Express.js
- MongoDB / PostgreSQL
- Multer (for image upload, if added)
- JWT (if auth is used)
- RESTful API

---

## ✨ Features

- 📝 Create and manage events
- 📅 View upcoming events with date, location, and organizer info
- 📸 Upload event images (optional)
- 📥 Register or RSVP to events
- 🎛️ Admin panel for event organizers
- 🔍 Search/filter events by category or location
- 📱 Fully responsive and mobile-ready UI
- ⚙️ REST API integration
